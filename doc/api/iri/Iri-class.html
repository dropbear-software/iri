Iri

Iri class

An Internationalized Resource Identifier (IRI).

An IRI is a sequence of characters from the Universal Character Set (Unicode). It provides a complement to the Uniform Resource Identifier (URI).

This implementation is a wrapper around the native Uri class, providing Seamless mapping to URIs as defined in RFC 3987.

All strings passed to constructors or parsing methods are automatically normalized to Normalization Form KC (NFKC).
